# CPU Scheduling Simulator

Interactive dashboard to visualize CPU scheduling algorithms with animated Gantt charts.

**Algorithms:** FCFS, SJF, SRTF, Priority, Round Robin, Multilevel Queue

## Setup

**Prerequisites:** [Node.js](https://nodejs.org/) (v18 or later)

```bash
# 1. Install dependencies
npm install

# 2. Start dev server
npm run dev
```

Open `http://localhost:5173` in your browser.

## Usage

1. Select an algorithm from the dropdown
2. Add processes manually or click **🎲 Random**
3. Click **▶ Start** to run the simulation
4. Use **Step Mode** for step-by-step execution
